Юниты

ЛКМ — выбор / рамкой   Shift+A — выделить всех

ПКМ по земле — движение   ПКМ по врагу — цель

a + клик — атака приближающегося врага   P + клик — патруль

S — разделить всех выделенных   M — слить выделенных

1–2 — режим (ручной / агрессор)

F — формация (плотно / линия / клин / кольцо)

Ctrl+1..5 — горячая группа   Shift+1..5 — вызов группы

Синтез (нижняя панель)

ЛКМ по кнопке с цифрой — поставить в очередь

ПКМ по кнопке — цикл LOOP (автоповтор)

Стоимость = значение² энергии. Энергия восстанавливается сама.

Типы юнитов:

1 Разведчик: быстрый, большой обзор, не наносит урон ядру

2 Рой: +10% урона за юнита роя(макс. +30%)

3 Щитоносец: броня 15%, медленный, щит ×1

4 Медик: лечит ядра (1/с) и юнитов (1/с)

5 Дробовик: урон по площади

6 Снайпер: дальность 5, мин. 3, стреляет снарядами

7 Делитель: при смерти разделяеться 2× со значением 3

8 Колосс: реген +0.25/с

9+ Сверхъюнит: растущие урон, броня, обзор и тд
